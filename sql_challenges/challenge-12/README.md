# SQL Challenge 05 – Union, Minus, and Intersect

## Ejercicios realizados

1. **UNION** – Listar colores únicos de ambas tablas
2. **UNION ALL** – Listar todas las formas (con duplicados)
3. **MINUS** – Listar formas en my collection que no están en your collection
4. **INTERSECT** – Listar colores presentes en ambas tablas

## Tecnologías
- SQL (Oracle / SQLite)
- Google Colab
- Python (matplotlib para gráficos)

## Resultados
- UNION: blue, green, red, yellow
- UNION ALL: circle, circle, circle, square, square, square
- MINUS: square
- INTERSECT: green, red
