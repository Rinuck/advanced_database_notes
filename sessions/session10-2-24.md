## Topics covered
Mathematical Aggregation
Using functions like AVG, SUM, MAX, and MIN to condense a full column of numbers into a single, meaningful value (such as a total or an average).

Data Counting
Using COUNT(*) to quickly determine the total number of records within a table or a specific filtered result.

Filtering Groups with HAVING
Using the HAVING clause to filter data after it has been grouped. This is essential because the WHERE clause cannot be used directly on aggregated results.
